# KMsWA2
This repository contains the implementaion of Krawtchouk Moments based Watermarking Attack, a black box attack on medical images.

![KMsWA2](media/KMsWA2.jpg)
## Dependencies
* Python (>=3.7)
* NumPy (>= 1.20.1)
* pillow (>=8.1.0)
